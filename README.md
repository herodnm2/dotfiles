Void Linux (Wayland).

## Setup

```bash
git clone https://github.com/herodnm2/dotfiles.git
cd dotfiles
chmod +x install.sh
./install.sh
```

## requirements

* alacritty
* waybar
* wofi / rofi
* gtk-3.0 / gtk-4.0
* qt5ct / qt6ct
* Kvantum
* swaync
* eww
* wal / matugen
* fish
* fastfetch
* btop
* msnap
* cava
* mango
